# 误差校准模块测试方案

## 📁 目录说明

这是一个独立的测试方案模块,用于验证误差校准模块的7大核心功能。

### 文件结构

```
calibration_test_plan/
├── README.md                           # 本文件,使用说明
├── 测试方案_误差校准模块验证.md         # 详细测试方案文档
├── test_data/                          # 测试数据存储目录(待创建)
│   └── YYYYMMDD/                      # 按日期组织
│       └── TEST_NAME/                 # 具体测试任务
│           ├── config.json            # 测试配置
│           ├── raw_data/              # 原始测量数据
│           ├── calibrated_data/       # 校准后数据
│           ├── temperature_log.csv    # 温度记录
│           └── test_log.txt           # 测试日志
├── calibration_data/                  # 校准系数存储(待创建)
└── test_reports/                      # 测试报告输出(待创建)
```

## 🎯 测试方案概述

### 验证的7大功能模块

1. **自动化控制** - 无人工干预完成测试流程
2. **参数配置** - VNA、开关矩阵、校准参数等配置管理
3. **指令下发** - SCPI指令正确下发与执行
4. **数据存储回放** - 测试数据的完整保存和历史数据回放
5. **数据采集** - S参数、功率、环境条件等多源数据采集
6. **全链路幅相误差修正** - 系统误差建模与修正算法
7. **不确定度评估** - 符合GUM规范的不确定度分析

### 测试对象

- **硬件系统**: 
  - VNA (矢量网络分析仪)
  - 开关矩阵
  - 校准模块
  - 功率计
  - 待测设备(频综收发机、功分器、TR组件)
  
- **软件系统**:
  - 集成测试控制软件
  - 误差修正算法
  - 数据处理与报告生成模块

- **测试环境**:
  - 实验室室温环境 (20-25°C)
  - 温度相对稳定，无明显波动

## 📖 使用方法

### 步骤1: 阅读测试方案
打开 `测试方案_误差校准模块验证.md` 详细阅读测试流程

### 步骤2: 准备测试环境
根据文档第2节"测试环境配置"搭建测试环境

### 步骤3: 执行测试
按照第3节"详细测试步骤"逐项执行:
- 3.1 自动化控制功能测试
- 3.2 参数配置功能测试
- 3.3 指令下发功能测试
- 3.4 数据存储与回放功能测试
- 3.5 数据采集功能测试
- 3.6 全链路幅相误差修正功能测试
- 3.7 不确定度评估功能测试

### 步骤4: 记录测试结果
在第6节"测试记录表"中填写测试结果

### 步骤5: 生成测试报告
根据第7节"测试结论"总结测试结果

## ⚠️ 注意事项

1. **隔离性**: 本测试方案与integrated_system其他模块隔离,不影响现有软件功能
2. **环境稳定性**: 测试期间保持实验室温度相对稳定,避免空调直吹、阳光直射等
3. **数据备份**: 重要测试数据及时备份,避免丢失
4. **设备稳定**: 测试前确保所有设备已经预热至少30分钟,达到热平衡状态
5. **校准周期**: 建议每次测试前进行一次系统校准，或至少验证校准有效性
